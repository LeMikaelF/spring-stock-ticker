# Stock Ticker

**Technologies used**
- Spring Boot (framework)
- JavaFX (UI)
- JavaFX-Weaver (Spring-FX bridge)
- REST API ([finnhub.io](https://finnhub.io/))

Demo application using Spring and a REST service to track stock prices.
